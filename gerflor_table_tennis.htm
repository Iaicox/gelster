Покрытие для настольного тенниса Taraflex Table Tennis от Gerflor

Taraflex Table Tennis Gerflor — это спортивное покрытие для настольного тенниса.

Спортивное покрытие Жерфлор Тарафлекс Тэйбл Теннис разработано в сотрудничестве с Международной федерацией настольного тенниса (ITTF) и Европейским союзом по настольном теннису (ETTU).

Покрытие для настольного тенниса Taraflex Table Tennis Gerflor легко устанавливается и монтируется.

Структурированное тиснение покрытия для настольного тенниса Taraflex Table Tennis от Gerflor обеспечивает удобство при игре.

Покрытие доступно в общей толщине: 5.0; 7.0 и 3.7 мм.

Дизайны покрытия для настольного тенниса имеют один артикул, но разные текстуры: есть  покрытие с "бугристой" поверхностью (слева), а справа покрытие с тисненением.
