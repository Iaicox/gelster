Ковровая плитка с рисунком Tessera Create Space 3

(Тессера Креэйт Спэйс 3)

Ковровая плитка Tessera Create Space 3 обладает интересным геометрическим рисунком, который позволит создать оригинальный дизайн Вашего помещения, оживив интерьер. Модульность коврового покрытия Tessera Create Space 3 позволяет комбинировать фрагменты, что поможет создать привлекательную композицию на поверхности пола, реализовав даже самые смелые идеи. 

Плитка ковровая Tessera Create Space 3 прекрасно сочетается с коллекциями Tessera Create Space 1 и Tessera Create Space 2, что увеличивает возможности декора и позволяет комбинировать большое количество цветов и различные стили покрытий. Применение сразу трех коллекций ковровой плитки с рисунком Тессера Спейс позволяет зонировать пространство, создавать эффект расширенного помещения.

Полосатый рисунок на ковровой плитке Tessera Space 3 позволяем укладывать ее в разных направлениях, создавая таким образом интересные композиции.

Tessera Create Space 3 является износостойким материалом, который не теряет своих свойств и не выцветает в процессе эксплуатации, за ним легко ухаживать, а специальная грязезащитная пропитка не позволяет попасть грязи и пыли в глубину покрытия.

Коллекция ковровой плитки с рисунком Forbo Tessera Create Space 3 обладает высокой прочностью и износостойкостью (класс применения 33), что делает ее актуальной для офисных помещений, переговорных и презентационных помещений, банкетных залов, гостиничных комплексов, помещений кинотеатров и других помещений с высоким уровнем проходимости.
