Ковровая плитка Desso Tree (Дессо Три)

Коллекция снята с производства, смотрите аналоги

Коллекция ковровой плитки Desso Tree (Дессо Три) – качественный ковровый материал в плитках размером 50х50 см. Для  стабильности размеров плитка имеет битумную основу. 

Оттенки ковровой плитки Desso Tree состоят из нежных тонов, которые сочетаются с богатой текстурой, создавая теплоту и комфорт.

Данная ковровая плитка отличается повышенной устойчивостью к нагрузкам, надежностью и долговечностью. Desso Tree (Дессо Три)  подойдет для мест с высокой проходимостью: гостиницы, санатории, офисы, рестораны, развлекательные центры бизнес-центры и т.п.
